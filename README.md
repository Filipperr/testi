# Å-Kollektiivi – Verkkosivusto

Tämä repositorio sisältää **Å-Kollektiivin** virallisen verkkosivuston lähdekoodin. Å-Kollektiivi on valokuvataiteilijoiden yhteenliittymä, joka toimii ilman hierarkiaa, tehden taidetta itsenäisesti ja yhdessä.

## 🎨 Visuaalinen ilme
Sivusto on suunniteltu minimalistiseksi ja ilmavaksi, heijastaen kollektiivin taiteellista luonnetta:
* **Typografia:** Puhdas sans-serif, jossa korostuu suuri ja näyttävä Å-kirjain.
* **Värimaailma:** Luonnonläheinen ja pehmeä (luonnonvalkoinen, hiili, hiekka).
* **Vastuullisuus:** Täysin responsiivinen design, joka skaalautuu puhelimesta työpöytäkäyttöön.

## 🛠️ Tekniikka
Sivusto on rakennettu "Single Page" -periaatteella käyttäen vakiotekniikoita:
* **HTML5:** Semanttinen rakenne ja saavutettavuus.
* **CSS3:** Modernit CSS-muuttujat, Flexbox, Grid ja dynaaminen typografia.
* **Lokalisaatio:** Sivusto on suomenkielinen.

## 📂 Sivun osiot
- **Manifesti:** Kollektiivin ideologia ja arvot.
- **Kollektiivi:** Lista mukana olevista taiteilijoista.
- **Yhteystiedot:** Suora linkki sähköpostiin.

## 🚀 Käyttöönotto
Koska kyseessä on staattinen sivu, se ei vaadi erillistä asennusta.
1. Kloonaa repositorio: `git clone [url]`
2. Avaa `index.html` selaimellasi.
